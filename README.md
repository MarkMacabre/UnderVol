# UnderVol VST3 Plugin

This is a VST3 plugin project for UnderVol.

## Project Structure

- **CMakeLists.txt**: Build configuration file for CMake.
- **Source/**: Contains the source code for the plugin.
  - **PluginProcessor.h**: Header file for the audio processor class.
  - **PluginProcessor.cpp**: Implementation of the audio processor class.
  - **PluginEditor.h**: Header file for the editor class.
  - **PluginEditor.cpp**: Implementation of the editor class.
- **Resources/**: Contains resources such as images and binary files.
